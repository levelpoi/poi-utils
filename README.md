# Agent Framework Design

一个用于设计和实现特定任务的 Agent 框架协作项目。

## 项目状态
🚧 **设计阶段** - 正在构思架构和核心逻辑

## 协作原则

### AI 权限
- ✅ 主动提出设计方案
- ✅ 质疑不合理的需求
- ✅ 挑战可能的技术陷阱
- ✅ 拒绝明显错误的方向
- ✅ 反问澄清模糊的需求

### 决策流程
1. 需求讨论 - 双方平等提出观点
2. 技术评估 - AI 提供可行性分析
3. 方案对比 - 列举利弊，共同决策
4. 迭代优化 - 持续挑战和改进
